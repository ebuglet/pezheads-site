Instructions: I think this is all pretty straight forward, but just in case...

  1. Make sure the first selection menu is set to how you want to search! If you are searching by name, type the name in the text box. Otherwise use the correct selection menu (that corresponds with what you're searching for).
  2. The "Country" menu is dynamically updated by the entries in the database. The database contains no other foreign pezheads other than what is in the menu.
  3. Searching by "Name" looks for the word you typed in both First and Last Name fields. If you can't spell the name you're looking for just type part of the name to narrow the search down.
