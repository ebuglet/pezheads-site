1. Purpose of this miniFAQ

The purpose of this FAQ is to answer the oft-asked question "What color does ----- come in?" for certain Pez U.S. dispensers. This FAQ does not intended to cover every color combination of every dispenser ever made. At least not right now.

This FAQ is intended to be a supplement to Chris Sharpe's Pez FAQ and is not meant to compete with it.

3. Abbreviations used in this FAQ

The following abbreviations are used in this FAQ for colors only:

  • T = Teal
  • R = Red
  • Bk = Black
  • P = Purple
  • Bu = Blue
  • O = Orange
  • Y = Yellow
  • G = Green
  • L = Lavender or dusty lavender rose (Peter Pez only)

6. Truck color combinations

Before the listing, here are a few words about trucks. There are five distinct classes of trucks, described below:

  • A - All wheels move (roll), one set of back wheels
  • B - All wheels move, 2 sets of back wheels w/ "m" shaped fender
  • C - All wheels move, 2 sets of back wheels
  • CR - All wheels move, 2 sets of back wheels, same cab styles as D Class
  • D - All wheels fixed, 2 sets of back wheels

The next list is for 4.9 stem, class "D" trucks (current release), and 3.9 stem, class "CR" trucks. Cabs are identified by the markings, R1, R2, R3, or R4 molded on each. A quick way to tell the difference between each cab is that R1 has the longest snout, and each getting progressively shorter, with R4 having a flat front end. (R3 is close but has a slightly bowed out front end.) There are four cab colors and three stem colors, but stem colors are never the same as the cab colors. There are thirty-six varieties.

Here is my suggestion for using the trucks list. Use 3b for a bagged 3.9, 3c for a carded 3.9, 4b for a bagged 4.9, 4c for a carded 4.9, 3 for a loose 3.9, and 4 for a loose 4.9. If you have a better idea, please let me know, and I'll add it to this FAQ AND give you credit. What a deal, eh?


7. Power Pez color combinations

The next list is for CAP Industries's Power Pez. It comes in three color schemes: O base with P spinner, G base with L spinner, and P base with G spinner. The bases colors are fluorescent. The turntables (which hold the Pez) are various colors.
