How does this thing work?

  • Actual PEZ purchases/sales are recorded into a huge data file of every dispenser ever made. It adds them all up and finds the average dollar amount spent.
  • When searching, you can choose which data files to use (in case you don't think one of the sources is accurate) and it will adjust it's values for you.
  • The data is updated on a regular basis by the people listed below. As time goes on, we'll keep getting more and more accurate values, as well as add more search features.
  • The "buy this dispenser" links search the One Stop PEZ Shop for possible matches for sale. Just kind of an added bonus to the site..

What features will the Price Guide have in the future?

  • After a substantial amout of new data gets entered, I'll add features to help you persoanalize your search. Search options such as voiding the top and bottom 5%, weighing newer entries more, using only the 5 most recent values, etc.
  • Although I think it's dumb, a lot of people seem to want a text list of the values so they can print it out. Seems to me that defeats the purpose of having a dynamic, regularly updated website...but I'll add that feature anyway.
  • Photos, photos, photos! (this has begun!)

How can I help this thing become better?

  • Any time you purchase a dispenser or PEZ item, email Kenneth Hooper and tell him so he can update the database. We NEED your help! Please include as much detail as possible when you write.. Head colors, patent numbers, base colors, feet/no feet, etc..
  • SUBMIT PICTURES! I know there are other sites out there that are collecting photos for similar things. But with this, I think there's a bit more info for collectors to use, along with the link to the One Stop PEZ Shop. I'd also like to make those dispensers for sale link to their respective images... So I'm looking for pictures of any and all discontinued dispensers.
